亞哥特色

離墾丁大街不到5分鐘車程

亞哥之家位於船帆石的海濱，開車到熱鬧的墾丁大街不用五分鐘，卻又可以遠離塵囂，保留夜間的寧靜。

180度無敵海景

亞哥之家大部分客房附陽台享有海景景觀，頂樓房型均有玻璃屋和按摩浴缸，讓您隨時隨地都能享受無際海洋、美麗日落。

交通方式

自行開車

1. 國道一號（中山高）→ 高雄小港 → 台17線 → 水底寮 → 台1線 → 楓港 → 車城 →台26線（屏鵝公路）船帆路 → 過墾丁福容飯店後，行駛約500公尺，亞哥之家就在您的左方。

2. 國道三號（二高）→ 屏東南州交流道 → 台1線 → 水底寮 → 台1線 → 楓港 → 車城 → 台26線（屏鵝公路）船帆路 → 過墾丁福容飯店後，行駛約500公尺，亞哥之家就在您的左方。

3. 東部（台9線) → 楓港 → 車城 → 台26線（屏鵝公路）船帆路 → 過墾丁福容飯店後，行駛約500公尺，亞哥之家就在您的左方。

飛機

1. 高雄國際機場 → 轉搭接駁計程車（共乘方式，可事先告知代訂）直達亞哥之家；或乘坐乘捷運至高鐵左營站 → 於2號出口轉搭 9188 墾丁快捷公車至船帆石前的消防局 → 步行前往亞哥之家。

2. 恆春機場 → 轉搭計程車直達亞哥之家或 9188 墾丁快捷公車至船帆石前的消防局（機場至墾丁約30分鐘車程）→ 步行前往亞哥之家。

高鐵

1. 高鐵左營站 → 轉搭接駁計程車 (共乘方式，可事先告知代訂）直達亞哥之家；或於2號出口轉搭 9188 墾丁快捷公車至船帆石前的消防局 → 步行前往亞哥之家。

火車

1. 高雄火車站 → 轉搭接駁計程車 (共乘方式，可事先告知代訂）直達亞哥之家；或於2號出口轉搭 9188 墾丁快捷公車至船帆石前的消防局 → 步行前往亞哥之家。

代訂接駁計程車

1. 高雄市區、火車站單趟1人NT400。

2. 左營高鐵站單趟1人NT450。

3. 高雄小港機場單趟1人NT500。
